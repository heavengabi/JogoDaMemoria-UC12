# 🧠 Jogo da Memória — Blue Period
#### https://heavengabi.github.io/JogoDaMemoria-UC12/
Projeto de um jogo da memória desenvolvido com foco em prática de JavaScript, manipulação do DOM e estilização com CSS. Inspirado no tema *Blue Period*, com uma paleta em tons de azul.

## 📌 Sobre o projeto

O jogo consiste em encontrar pares de cartas iguais, testando memória e atenção do jogador.

## 🚀 Tecnologias utilizadas

- HTML
- CSS 
- JavaScript 

## 🧩 Conceitos aplicados

- Manipulação do DOM  
- Eventos em JavaScript  
- Controle de estado (lógica do jogo)  
- Estilização com CSS  
- Layout com display flex  

## 🎨 Design

O visual do jogo foi inspirado no anime/mangá Blue Period, utilizando:

- Tons de azul  
- Interface simples e intuitiva  
- Organização com Flexbox  

## 🕹️ Como jogar

1. Clique em uma carta para revelar  
2. Clique em outra carta para tentar encontrar o par  
3. Se forem iguais, elas permanecem viradas  
4. Se forem diferentes, voltam ao estado inicial  
5. Continue até encontrar todos os pares
